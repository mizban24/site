## 💡 V2Ray چیست؟
V2Ray یک پروژه متن‌باز برای ایجاد ارتباطات خصوصی و ایمن است.  
پشتیبانی از پروتکل‌های **VMess, VLESS, Trojan** و بهترین ابزار برای عبور از سانسور اینترنت.
